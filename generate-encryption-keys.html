📋 How to Add These Keys to Replit:

  1. Generate the keys above by clicking the button
  2. Copy each key using the "Copy Key" buttons
  3. Go to your Replit project
  4. Click the lock icon (🔒) in the left sidebar to open Secrets
  5. Add a new secret named MEDICAL_DATA_ENCRYPTION_KEY and paste the first key
  6. Add another secret named MEDICAL_DATA_SALT and paste the second key
  7. Your server will automatically restart with encryption enabled
